# Conversion Partners Network - Brand Guidelines

## Color Palette

This site uses a **black and off-white** color scheme:

- **Background**: White (`#ffffff`)
- **Foreground/Text**: Black (`#111111`)
- **Muted backgrounds**: Off-white/light gray (`#f8f8f8`)
- **Muted text**: Gray (`#6b7280`)
- **Accent (sparingly)**: Teal (`#1e7b85`) - used for links and subtle accents only

## Design Principles

1. **Buttons**: Always use black (`bg-foreground text-background`) for primary buttons
2. **Modals/Dialogs**: Use `bg-background` (white) with `text-foreground` (black)
3. **Inputs**: White background with subtle gray borders (`border-muted-foreground/20`)
4. **Typography**: Clean, minimal - Inter font family
5. **Spacing**: Generous whitespace, clean layouts
